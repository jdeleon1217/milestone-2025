Hello! This is project for my website called VisitCastles website which focus on giving information about Swiss castles for touristic purposes.

In this project, you will find the following files and folders below:

1) "index.html" which contains the code for the homepage of the website
2) "info.html" which contains the code for Contact details and a form for a newsletter
3) "about.html" which contains the code for the details and staff of the company
4) "styles.css" which is the code for every style, color and patterns for all pages
5) "Castle" folder which holds files of the different castles
6) "Images" folder which contains images used for all pages
7) Finally, "js_visitcastle" folder which hold functionality to all pages: modals, dark or light mode, image sliders, modal box, category filter and smooth scrolling
